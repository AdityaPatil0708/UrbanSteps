# sneaker-store

This is a full stack project built with MERN stack. It is a sneaker store website where users can browse, buy and sell sneakers.

## Features
- User authentication
- User registration
- User verification
- User login
- User logout
- User profile
- User profile update
- User profile delete
- User profile image upload
- User profile image delete
- User profile image download
- User profile image view
- User profile image view all
- User profile image view by id
- User profile image view by user id
- User profile image view by sneaker id
- User profile image view by category id
- User profile image view by brand id
- User profile image view by gender id
- User profile image view by price id
- User profile image view by availability id
- User profile image view by material id
- User profile image view by pattern id
